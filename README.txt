# Algorithms_lab_iot

Rabin-Karp algorithm slides the pattern one by one. 
Rabin Karp algorithm matches the hash value of the pattern with the hash value
of current substring of text, and if the hash values match then only it starts matching individual characters

how to  launch algorithm
1.Download zip file from the branch
2. Via CMD go to root folder
3.type "pyton main.py" and here are you go
